Installation

For a copy of LCS Toolbox, contact Kristjan Onu with a brief description of your interests in LCS Tool. You will need to provide an SSH public key. The following commands will provide a copy. Git must be installed.

  1. git clone gitolite@lyapunov.mecheng.mcgill.ca:lcs_toolbox
  2. cd lcs_toolbox
  3. git submodule init
  4. git submodule update
Once LCS Tool has been copied, updates can be obtained with the following commands. They must be executed inside the LCS Tool folder.
  1. git pull origin master
  2. git submodule update

Documentation

LCS Tool works with MATLAB. To access documentation for LCS Tool, start MATLAB, then type doc at the prompt. At the bottom of the page, click on Supplemental Software, then look for the LCS Tool User's Guide.
