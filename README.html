The LCS Toolbox

The LCS Toolbox performs computations for the analysis of Lagrangian coherent structures. It is based on work of George Hallerâ€™s Nonlinear Dynamics Group.

Installation

For a copy of the LCS Toolbox, contact Kristjan Onu. You will need to provide an SSH public key. The following commands will provide a copy of the toolbox. Git must be installed.

  1. git clone gitolite@lyapunov.mecheng.mcgill.ca:lcs_toolbox
  2. cd lcs_toolbox
  3. git submodule init
  4. git submodule update
Once the LCS Toolbox has been copied, updates can be obtained with the following commands. They must be executed inside the LCS Toolbox folder.
  1. git pull origin master
  2. git submodule update

Documentation

The LCS Toolbox works with MATLAB. To access documentation for the LCS Toolbox, start MATLAB, then type doc at the prompt. At the bottom of the page, click on Supplemental Software, then look for the LCS Toolbox User's Guide.
