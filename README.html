Installation

For a copy of LCS Toolbox, contact Kristjan Onu with a brief description of your interests in LCS Tool. You will need to provide an SSH public key. The following commands will provide a copy. Git must be installed.

  1. git clone gitolite@lyapunov.mecheng.mcgill.ca:lcs_toolbox
  2. cd lcs_toolbox
  3. git submodule init
  4. git submodule update

Once LCS Tool has been copied, updates can be obtained with the following commands. They must be executed inside the LCS Tool folder.

  1. git pull origin master
  2. git submodule update

Demo scripts

MATLAB scripts demonstrating the use of LCS Tool are in the folder named demo. To run these scripts, start MATLAB in the LCS Tool folder, then at the MATLAB prompt type, for example:

  1. cd demo/double_gyre/
  2. addpath ../..
  3. matlabpool open
  4. flow_animation

The time required to execute some of the demo scripts is given below. These times were measured on a laptop with an Intel Core i5-3360M processor with 16 gigabytes of memory:

  • double_gyre/hyperbolic_shear_lcs_details: 11 minutes
  • bickley_jet/stretchlines: 26 minutes
  • ocean_dataset/demo_script_ocean: 18 minutes
